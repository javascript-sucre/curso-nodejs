
## NODEMON

```sh
# instalar nodemon
$ npm install nodemon --global

# ejecutar un archivo con nodemon
$ nodemon file.js
```

## FOREVER

```sh
# instalar forever
$ npm install forever -g

# ejecutar un archivo con forever
$ forever start file.js
```
